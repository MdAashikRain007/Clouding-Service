# Clouding Service ☁️

A **Full Stack Cloud File Sharing Application** that allows users to upload, store, manage, and share files securely through a web interface.

The platform provides a **dashboard for file management, sharing functionality, and secure cloud storage integration**.  
It demonstrates full-stack development using **Java Spring Boot and React**.

---

# 🚀 Features

- ☁️ Cloud-based file storage system
- 📤 Upload files to the cloud
- 📥 Download stored files
- 🔗 Share files using secure links
- 🗂️ File management dashboard
- 👤 User authentication system
- 📊 Modern and responsive UI
- 🌐 REST API based backend architecture
